# 🎓 Campus Companion Agent

AI-powered campus assistant for academics, events, and smart permission management.

## 🏆 Microsoft Agents League Hackathon 2026 — Enterprise Agents Track

## 📌 Problem Statement
Students struggle to get information about academics, events, and permissions. Traditional permission processes involve multiple forms, signatures, and delays.

## 💡 Solution
Campus Companion Agent is an AI-powered assistant that provides:
- 🤖 **AI Chat** — instant answers on attendance, syllabus, exam queries
- 📅 **Event Updates** — real-time campus event board
- 💼 **Career Guidance** — internship tips, placement prep
- 📋 **Smart Permission Management** — submit leave/OD requests conversationally

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript
- Anthropic Claude API (claude-sonnet-4-6)
- GitHub Pages (hosting)
